Sync Engine views Folder

This folder contains site-specific settings for database views

The 2 main files in this folder are the scripts to create the Difference views used for change detection, and the Publication views that are used to filter out sensitive data and also include additional fields from available tables.

There are also some other scripts in this folder that are used to build indexes, perform some QA functions, and use SQL to update geometry Z values.
