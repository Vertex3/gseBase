--- title: 'runSyncSites.py script' tagline: Floorplan Sync Engine layout: default published: true --- ## Doc for runSyncSites.py script
Use the sitesCAD.txt file for the list of sites to process For each site:
  • Run bldgLoadPlaylist.xml to load Bldg Outlines
  • Run fpLoadPlaylist.xml to load floorplans
  • Run rtLoadPlaylist.xml to load Routing drawings
