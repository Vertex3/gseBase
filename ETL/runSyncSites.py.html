Documentation for runSyncSites.py scriptThis process is the main synchronization process that is run on a regular basis as a scheduled task. The process typically starts at the top of each hour and processes and drawings from the CAD Vault mirror that are present since the last update.

Use the sitesCAD.txt file for the list of sites to process

For each site:

  • Run bldgLoadPlaylist.xml to load Bldg Outlines
  • Run fpLoadPlaylist.xml to load floorplans
  • Run rtLoadPlaylist.xml to load Routing drawings
  • Run runRoutingExtract.py to extract changed routing drawings by building since this process started
